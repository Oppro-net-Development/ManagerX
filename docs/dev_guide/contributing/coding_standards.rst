Coding Standards
================

To maintain high code quality, please follow these guidelines:

Python Style
------------
* **PEP8**: Follow the PEP8 style guide strictly.
* **Naming**: Use meaningful variable and function names.
* **Docstrings**: Add docstrings for all public functions and classes.

Design Principles
-----------------
* Write **modular** and reusable code.
* Ensure **backward compatibility** wherever possible.
* Keep functions small and focused on a single task.
